## Variant: MPC-Workbench dense

### Design stance
Pad-first Werkbank. Das Instrument ist die Fläche — Chat ist Dock, nicht Bühne.

### Key choices
- Layout: 4×4 Pads Zentrum, Library links, Inspector rechts, Transport oben, Agent-Dock unten
- Typography: Inter / system mono für BPM/Key
- Color: near-black chassis, amber pad hit, cold cyan data
- Interaction: Pads spielen Web-Audio; Steps togglen; Chat-Chips füllen Pads

### Trade-offs
- Strong at: sofortiges Greifen, Beatmaker-Feeling, geringe kognitive Last
- Weak at: Arrangement/Song-Sicht fast unsichtbar

### Best for
Chop → Pattern → Loop. Hip-Hop / electronic sketching. Primär-Richtung wenn „MPC“ der emotionale Kern ist.
