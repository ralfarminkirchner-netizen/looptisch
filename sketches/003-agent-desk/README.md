## Variant: Agent-Desk

### Design stance
Tisch-Metapher: Crew-Rollen sichtbar, jede Aktion ist ein Werkzeug am Projekt — Chat steuert echte State-Änderungen.

### Key choices
- Layout: oben Crew-Chips, Mitte Pads+Sequenz als „Werkstück“, unten Konsole mit Tool-Calls
- Typography: readable workbench, mono for tool log
- Color: warm paper-dark / brass / signal green for agent success
- Interaction: agent buttons + chat; tool log shows API calls (fill_pads, set_stretch…)

### Trade-offs
- Strong at: macht das KI-Versprechen glaubwürdig (sichtbare Agency)
- Weak at: kann nach „Dashboard“ riechen wenn Pads nicht dominant genug bleiben

### Best for
Demoing der Produkt-These „Agenten bewegen Musik, nicht nur Text“. Gut für Pitch + Spike-Spec.
