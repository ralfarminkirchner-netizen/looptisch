## Variant: Pattern-Flow

### Design stance
FL-Studio-DNA: Patterns als Zeilen, Playlist als Filmstreifen, KI als rechte Schiene.

### Key choices
- Layout: Channel-Rack links/mitte, Playlist oben/mitte, AI-Rail rechts
- Typography: compact UI sans, dense numbers
- Color: graphite + fruit-orange accent (nicht FL-Klon, nur Energie)
- Interaction: Step-cells, clip blocks drag-feel (click place), agent actions on selection

### Trade-offs
- Strong at: Song-Struktur, mehrere Patterns, Remix-Überblick
- Weak at: Pad-Performance fühlt sich sekundär an

### Best for
Leute die in Loops denken und schnell Arrangement wollen — „Cubase light timeline“ ohne Menüwald.
