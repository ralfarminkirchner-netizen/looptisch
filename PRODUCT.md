# LOOPTiSCH

> Cooperative Beat Instrument — Körper greifen, Crew erledigt Komplexität.

## Scene
Studio-Chassis. Mensch spielt Pads. Agenten zerlegen Studio-Arbeit in freigabefähige Schritte.

## Product pillars
1. **Körper** — Pads, Seq, Arrange, Chop (ohne KI spielbar)
2. **Crew** — Missionen + Multi-Step-Pläne + Optionskarten
3. **Undo** — jede Mutation umkehrbar
4. **Laien-Mühelosigkeit** — komplexe Vorgänge als 1-Tap-Missionen
5. **Project-API only** — LLM mutiert nur über Whitelist-Tools

## Primary actions
- Mission starten (First Beat, Break→Kit, Drop, Genre, Fix Mix, Remix)
- Option A/B wählen
- Pads greifen / Undo
- Optional: freier Chat an die Crew

## Non-goals
Full DAW clone · silent agent overwrites · score editor as core
